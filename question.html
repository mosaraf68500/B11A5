Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML:Gets or sets the HTML content (tags included) inside an element.
innerText: Gets or sets the visible text inside an element (ignores HTML tags).
textContent: Gets or sets all text inside an element, including hidden text (ignores styling).

Question-3: What is event delegation in the DOM?

Event delegation হলো এমন একটি পদ্ধতি, যেখানে একাধিক উপাদানের জন্য আলাদা আলাদা ইভেন্ট লিসেনার না দিয়ে, একটিমাত্র প্যারেন্ট উপাদানে ইভেন্ট লিসেনার যোগ করা হয়। এরপর, প্যারেন্ট উপাদান চাইল্ড উপাদানগুলোর ইভেন্টগুলো ক্যাচ করে।

Question-4: What is event bubbling in the DOM?

Event bubbling হলো ডকুমেন্ট অবজেক্ট মডেল (DOM)-এ ইভেন্টের একটি আচরণ। যখন কোনো ইভেন্ট (যেমন, ক্লিক) একটি উপাদানে ঘটে, সেই ইভেন্টটি প্রথমে সেই উপাদানে ঘটবে, তারপর ধীরে ধীরে ওই উপাদানের প্যারেন্ট (মা) উপাদানে এবং তারও উপরের উপাদানগুলোর দিকে বুবল (উত্তোলিত) হবে। এই প্রক্রিয়া চলে যতক্ষণ না এটি ডকুমেন্ট অবজেক্ট মডেলের (DOM) রুটে পৌঁছায়।
